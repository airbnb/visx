@vx/group

A <Group /> is a container for other objects. It lets you pass in a top and left margin and a classname.

Installation

npm install --save @vx/group

Components

  • Group

API

<Group />

# Group.children<any>

# Group.className<string>

# Group.innerRef<union(func|object)>

# Group.left<number>
Default0

# Group.top<number>
