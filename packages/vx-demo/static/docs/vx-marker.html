@vx/marker

A Marker is a line with a piece of text attached to it. It's great for highlighting locations in your graph.

Example

<Marker
  from={markerFrom}
  to={markerTo}
  stroke={'white'}
  label={'deploy'}
  labelStroke={'none'}
  labelDx={6}
  labelDy={15}
/>

Installation

npm install --save @vx/marker
