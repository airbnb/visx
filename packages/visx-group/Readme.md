# @visx/group

`<Group />` provides a simplified API for SVG `<g />` elements, which are containers for other SVG
objects. You may pass in a `top` and `left` margin (instead of `transform={translate(...)}`) and a
`className`.

## Installation

```
npm install --save @visx/group
```
